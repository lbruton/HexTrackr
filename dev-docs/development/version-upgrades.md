# Version Upgrades

This document provides guides for upgrading to new versions of the HexTrackr application.

---

## Upgrading from v1.x to v2.x

Version 2.0 includes breaking changes to the database schema. Follow these steps to upgrade:

1.  **Backup**: Create a backup of your v1.x database.
2.  **Install**: Install the v2.x application.
3.  **Migrate**: Run the database migration scripts to update your database to the v2.x schema.
4.  **Test**: Test the application to ensure that the upgrade was successful.
