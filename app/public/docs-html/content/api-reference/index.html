API Reference

This section provides detailed documentation for the HexTrackr REST API endpoints. All endpoints use standard REST conventions with JSON request/response bodies.

API Collections

  • Tickets API: Endpoints for creating, reading, updating, and deleting tickets.
  • Vulnerabilities API: Endpoints for managing and analyzing vulnerability data, including the historical trend analysis.
  • Backup and Restore API: Endpoints for performing system-wide backup and restore operations.
