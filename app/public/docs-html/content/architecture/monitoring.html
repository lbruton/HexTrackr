Monitoring

This document provides a guide to monitoring the HexTrackr application.

Health Check

The application provides a health check endpoint at /health. This endpoint can be used to monitor the status of the application.

Logging

The application logs to the console. You can use a log aggregation tool to collect and analyze the logs.

Metrics

The application does not currently expose any metrics. However, you can use a tool like Prometheus to collect metrics from the application.
