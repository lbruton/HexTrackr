Accessibility

This document describes the accessibility features of the HexTrackr application.

WCAG Compliance

HexTrackr is designed to be compliant with the Web Content Accessibility Guidelines (WCAG) 2.1 AA standard.

Accessibility Features

  • Keyboard Navigation: The application is fully navigable using a keyboard.
  • Screen Reader Support: The application is compatible with screen readers.
  • Color Contrast: The application uses a color palette that meets the WCAG AA color contrast ratio.
