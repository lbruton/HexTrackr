Version 1.0.91 - Fixed Version Display Accuracy Fix (HEX-299)

Release Status: ✅ Released
Release Date: 2025-10-19
Bug Fix: HEX-299 (Remove "+" suffix from Fixed Version displays)

Overview

Removed outdated "+" suffix from all Fixed Version displays throughout the application. The "+" was originally added to indicate "this version or higher" when displaying the first fixed version, but became misleading after the system evolved to display the latest/matched fixed version based on installed OS using major.minor family matching.

Bug Fixed

Outdated "+" Suffix in Fixed Version Displays

Root Cause: Display layer still appended "+" suffix from original implementation when system showed minimum fix version
Current Behavior: System now displays specific recommended version (e.g., "10.2.0-h3") using intelligent version matching
Issue: The "+" suffix (e.g., "10.2.0-h3+") implied "any version 10.2.0-h3 or later", causing potential confusion
Fix: Removed "+" suffix from all 8 display locations across 5 frontend files
Impact: Fixed version displays now show clean, specific version numbers matching database storage format
