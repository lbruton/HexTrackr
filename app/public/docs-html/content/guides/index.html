User Guides

  • Getting Started Guide: Quick start guide to get HexTrackr up and running in your environment, covering Docker installation, HTTPS setup, SSL certificates, database initialization, and troubleshooting

Customization

  • CSS Customization Guide: Customize HexTrackr's appearance to match your organization's branding including brand colors, custom logo, custom fonts, component styling (cards, buttons, badges, navigation), CSS variables, and theme compatibility
