## Notes

### Migration Pattern

**Frontend Utility Logging Pattern**:
```javascript
// BEFORE
console.log('Progress updated:', percent);
console.warn('Modal already open');
console.error('Failed to load settings:', error);

// AFTER
logger.debug('ui', 'Progress updated:', { percent });
logger.warn('ui', 'Modal already open');
logger.error('ui', 'Failed to load settings:', { error: error.message });
```

**Modal Lifecycle Logging**:
```javascript
// Modal initialization
logger.info('ui', 'Progress modal initialized');

// User interaction
logger.debug('ui', 'Settings updated', { setting: key, value });

// Error handling
logger.error('ui', 'Modal failed to open', { error: err.message, context: 'settings' });
```
