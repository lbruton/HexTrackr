HexTrackr is a comprehensive vulnerability and ticket management system designed for security teams. It provides centralized tracking of security vulnerabilities, managing support tickets, and integrating with various security tools and workflows.

## Key Features

- **Centralized Vulnerability Tracking**: Manages and tracks vulnerabilities from various sources.
- **Integrated Ticketing System**: A built-in system for incident response and tracking.
- **RESTful API**: Allows for automation and integration with other tools.
- **Docker-Based Deployment**: Ensures easy and consistent deployment.
- **SQLite Database**: Provides a lightweight and file-based database solution with backup and restore capabilities.
- **Modern UI**: Utilizes the Tabler.io and Bootstrap frameworks for a responsive and user-friendly interface.
- **Data Visualization**: Includes interactive charts and graphs for vulnerability trending and analysis.
- **Rollover Architecture**: An advanced feature for managing and tracking vulnerability data over time, providing historical trending and analysis.
